Necə işləyir?

Satış modeli üçün ideal: müqavilə → aktivləşmə → nəzarət

1
Müqavilə & Panel aktivləşməsi

Şirkət admini kursu “active” edəndən sonra kurs rəhbəri panelə daxil olur.

2
Manual qeydiyyat

Kurs admin müəllim/tələbəni əlavə edir, sistem random ID/şifrə yaradır.

3
Dondurma rejimi

Ödəniş gecikərsə şirkət admin “suspend” edir — data qalır, əməliyyatlar bloklanır.
